對 2019 的期許

2019-01-01
/ 2019

去年 2018 年對於自己的學習評價算是滿意，花了約一個月的時間熟練了 Vue，以及大大小小 SE 的一些事情和工具的安裝及使用，雖然本身並不是這個職位，但我認為這並不該是局限於 SE 該做的事情。而對於自己 project 的 CI/CD 流程也提昇了不少，因為今年在一場社群演講聽到了也有在掃引入 3-party lib 的一些免費或收費的軟體來加入 CI/CD 裡面，原來當你個別使用這些 lib 的時候他們在法律上是可以來當做商業用途的，但當一起的時候就會有衝突的可能，算是收穫之一，那這些東西就是來避免有法律糾紛的問題也是很重要的。除了期許今年可以做更多的 side project 出來，也列一些我在今年度希望可以學習或是達成的目標！

nuraphone 開箱

2018-12-25
/ nura

最近因緣際會透過朋友購買到了之前在 Kickstarter 上幕資達到 180 萬美金的 nuraphone，金額也達到了歷史紀錄。而 nura 不僅僅是一個耳罩式耳機，入耳的部份還是耳塞式，而透過耳罩處理低音頻，反之耳塞處理中高頻，也有方便的切換 社交模式 以及 降噪模式，令使用者在工作場合也可以自由切換是否需要聽到隔壁同仁的聲音，在戴上的那一刻，耳機就會針對您的頻率透過 nura app 來創建出一開始的聽覺設定。

重新整頓部落格

2018-11-25
/ blog prettypure

以前為了偷懶和方便，都是簡單架個快速的 blog, 例如 hexo, 在最近比較有空閒的時間，就想把 blog 整個重新整頓一下，也會找時間把以前的文章都補上。目前算是把 blog 給重新建置到一個段落，之後會把文章分頁給補一下應該就差不多了；而主題樣式一時間並沒有什麼想法，所以就大至先參考我的好兄弟 刀疤說 的 Nex ，而目前的主題我把它取為 prettypure 等有多餘的時間會在替換自己所喜愛的元素。
